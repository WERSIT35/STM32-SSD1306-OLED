# STM32 SSD1306 OLED Example

This project shows how to drive an SSD1306 128x64 OLED display via I2C on an STM32L031 using Keil.

## Features
- I2C1 (PB6=SCL, PB7=SDA)
- Custom SysTick delay
- 5x7 font for text rendering

## Connections
- OLED VCC → 3.3V
- OLED GND → GND
- OLED SCL → PB6
- OLED SDA → PB7
